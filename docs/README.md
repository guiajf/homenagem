# Vias denominadas em homenagem a Pelé em Três Corações/MG

Este diretório contém um mapa interativo contendo as vias urbanas de Três Corações,
denominadas em homenagem a Edson Arantes do Nascimento, Pelé, em destaque.

## Arquivos
- `index.html` - Página principal com o mapa
- `homenagem.html` - Mapa interativo gerado pelo Python
